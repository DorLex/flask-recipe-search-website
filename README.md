### Описание:

Сайт с интерактивным поиском кулинарных рецептов по имеющимся у вас ингредиентам.

### Запуск:

1. Создать файл `.env` по примеру `example.env`.

2. Запустить БД:
   ```shell
   make up-db
   ```

3. Проинициализировать БД:
   ```shell
   make init-db-data
   ```

4. Запустить сервер:
   ```shell
   make server
   ```
